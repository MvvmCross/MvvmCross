NOTE 
- Use Xamarin Studio for this...

0. Add a Xamarin.Mac Project
1. Add a reference to your Core PCL project, e.g., <root>.Core
2. Right-click on the project and under Tools -> Edit File, then add <TargetFrameworkVersion>4.5</TargetFrameworkVersion> in the first PropertyGroup, this is needed to access PCLs using Profile 158
3. Use nuget to add the MvvmCross.Mac package
4. Verify the Setup.cs file (automatically created)
5. Replace the contents of AppDelegate.cs/.fs from AppDelegate.cs/.fs.txt
6. Delete MainWindow.cs, MainWindow.xib and MainWindowController.cs
7. Create a Views folder and add a Cocao View and Controller, if the ViewModel is "MainViewModel", recommended naming for the View is "MainView"
8. Edit MainView.cs/.fs by adding the using/open statement below and changing the base type to MvxView

	C#
        using MvvmCross.Binding.Mac.Views;

        public partial class MainView : MvxView

	F#
        open MvvmCross.Binding.Mac.Views

        type MainView()
	    inherit MvxView()

9. Edit MainViewController.cs/.fs by adding the using/open statements below, adding MvxViewFor and changing the base type to MvxViewController

	C#
        using MvvmCross.Mac.Views;
        using MvvmCross.Binding.BindingContext;
        using MvvmCross.Core.ViewModels;
        using <root>.Core.ViewModels;

        [MvxViewFor(typeof(MainViewModel))]
        public partial class MainViewController : MvxViewController

	F#
        open MvvmCross.Mac.Views
        open MvvmCross.Binding.BindingContext
        open MvvmCross.Core.ViewModels
        open <root>.Core.ViewModels

        [<MvxViewFor(typeof<MainViewModel>)>]
        type MainViewController() =
	    inherit MvxViewController()

10. In MainViewController.cs/.fs, override ViewDidLoad, add controls programmatically or in the .xib, then bind, e.g.,

	C#
        public override void ViewDidLoad ()
        {
            base.ViewDidLoad ();

            ...
            
            var set = this.CreateBindingSet<MainViewController, MainViewModel> ();
            set.Bind (textEditFirst).For(v => v.StringValue).To (vm => vm.FirstName);
            set.Bind (textEditSecond).For(v => v.StringValue).To (vm => vm.LastName);
            set.Bind (labelFull).For(v => v.StringValue).To (vm => vm.FullName);
            set.Bind (bu).For("Activated").To ("GoCommand");
            set.Apply ();
        }

	F#
	override this.ViewDidLoad() =
	    base.ViewDidLoad()
	    ...
 

Where this requires using/open:

<core>.Core.ViewModels
MvvmCross.Binding.BindingContext
MvvmCross.Mac.Views
